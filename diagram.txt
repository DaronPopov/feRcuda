feR-os (feRcuda) - Architecture Overview
=======================================

User / App Layer
----------------
  Rust apps (fercuda-ffi, candle-ptx-os)   C/C++ apps   Python/FFI
           |                                    |             |
           +------------------------------------+-------------+
                                |
                         C API: libfercuda_capi.so
                                |

Runtime / Control Plane
-----------------------
  RuntimeSession
    |- JobManager
    |- ExecPlanner
    |- RegimeManager
    +-> ElasticPool (delegates to native TLSF)
          |- mutable (TLSF via gpu_hot_alloc/free)
          +-> immutable (append-only weights)

Native PTX-OS Layer (libptx_core.so, libptx_kernels.so)
-------------------------------------------------------
  GPUHotRuntime
    |- TLSF allocator (O(1) alloc/free)
    |- Stream pool
    |- Tensor ops (add, mul, matmul, softmax, reduce, ...)
    +-> JIT (intent lowering, fusion, PTX cache)

Memory Plane
------------
  ElasticPool (delegates to native)
    |- mutable slab (activations/scratch) -> gpu_hot_alloc/free
    |- immutable slab (weights/constants)
    +-> TLSF reclaim path (fast alloc/free reuse)

Compute Plane
-------------
  CUDA ops (native + bridge)
    |- elementwise (ptx_tensor_*)
    |- matmul, layer_norm (ops_bridge)
    |- activation, reduce, softmax
    +-> JIT intent execution

Integration / Compatibility
---------------------------
  libptx_hook.so (LD_PRELOAD optional)
    |- intercept cudaMalloc/cudaFree (+ async/managed/host/pitch)
    |- PTX_HOOK_MODE: tlsf | cuda | hybrid
    |- PTX_HOOK_VERBOSE, PTX_HOOK_DEVICE, PTX_HOOK_DISABLE
    +-> routes allocations through PTX-OS TLSF

Build + Install + Update
------------------------
  One-line installer
    curl .../scripts/install.sh | bash
      -> managed source checkout (~/.local/share/fer-os/src/feRcuda)
      -> update to latest origin/main on rerun
      -> CMake build/install
      -> Rust bootstrap (cargo fetch for crates)

Outputs
-------
  libfercuda.a (static)
  libptx_core.so
  libptx_kernels.so
  libptx_hook.so (intercept, LD_PRELOAD)
  libfercuda_capi.so
